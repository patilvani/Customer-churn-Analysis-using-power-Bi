📊 Customer Churn Analysis

Tools Used: Microsoft Excel | Power BI

📌 Project Overview

Customer retention is a key driver of business profitability. This project analyzes customer data to identify patterns and factors contributing to customer churn.

Using Microsoft Excel for data preprocessing and Power BI for visualization, the project delivers actionable insights through an interactive dashboard that helps businesses monitor churn trends and improve retention strategies.

🎯 Objectives

Identify key factors influencing customer churn

Analyze high-risk customer segments

Calculate churn-related KPIs

Develop an interactive dashboard for business decision-making

🛠 Tools & Technologies

Microsoft Excel

Data cleaning and preprocessing

Handling missing values

Removing duplicates

Creating calculated columns

Power BI

Data modeling

DAX measures

KPI development

Interactive dashboard creation

📊 Key Performance Indicators (KPIs)

Total Customers

Active Customers

Churned Customers

Churn Rate (%)

Revenue Impact

Customer Segmentation by Contract Type

🔎 Analysis Performed

Churn rate analysis by:

Contract Type

Tenure

Monthly Charges

Payment Method

Customer segmentation to identify high-risk groups

Revenue comparison between churned and retained customers

💡 Key Insights

Customers with month-to-month contracts have higher churn rates

Customers with short tenure are more likely to leave

Higher monthly charges increase churn probability

Long-term contracts improve customer retention

📈 Business Impact

This dashboard enables stakeholders to:

Monitor churn trends in real-time

Identify high-risk customer segments

Develop targeted retention strategies

Reduce revenue loss caused by customer attrition
